# 光影 LUMIERE - 个人摄影作品集网站

极简主义黑白摄影作品集网站，支持深色/亮色主题切换。

## 功能特性

### 前台页面
- **首页** - 全屏 Hero 区域配粒子连线动画、精选作品集入口
- **作品集** - 四个摄影栏目（城市建筑、旷野遐想、建筑光影、生活碎片），不对称网格布局
- **关于我** - 头像、个人简介、摄影履历数据
- **联系方式** - 留言表单、社交链接

### 后台管理
- **账号安全** - 修改登录密码
- **站点信息** - 网站名称、标题、描述、作者信息
- **作品集管理** - 分类增删改、照片上传、编辑、批量删除、排序
- **页面文案** - 首页、作品集、关于我、联系页、页脚所有文案
- **联系方式** - 自定义联系方式（微信、微博、小红书等）
- **留言管理** - 查看访客留言、标记已读、删除

### 其他功能
- 暗色/亮色主题切换（自动记忆用户偏好）
- 响应式设计，适配桌面和移动端
- 照片懒加载
- 优雅的暗色极简 UI

## 技术栈

- **后端**: PHP + SQLite (PDO)
- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **字体**: Cormorant Garamond (标题) + Inter/Noto Sans SC (正文)

## 快速开始

### 1. 启动服务器

```bash
cd /workspace
php -S 127.0.0.1:8000
```

### 2. 初始化数据库

首次访问 `http://127.0.0.1:8000/setup.php` 或运行：

```bash
php setup.php
```

### 3. 访问网站

- 前台首页: http://127.0.0.1:8000/
- 后台管理: http://127.0.0.1:8000/login.php

### 4. 默认账号

- 用户名: `admin`
- 密码: `admin123`

> 首次登录后请立即修改密码！

## 自定义

### 添加新分类

后台 → 作品集 → 添加分类（填写标题、slug、描述）

### 管理照片

后台 → 作品集 → 照片列表（点击选中 → 批量删除；点击编辑按钮 → 修改信息）

### 修改页面文案

后台 → 页面文案 → 对应页面表单

### 添加联系方式

后台 → 联系方式 → 添加（标签、值、链接）

## 注意事项

- 数据库使用 SQLite，文件位于 `database.sqlite`
- 上传图片时请使用完整的 URL 地址
- 建议定期备份 `database.sqlite` 文件
